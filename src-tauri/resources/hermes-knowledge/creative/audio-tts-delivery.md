# 台词配音：表演层提示词（从业者技巧）

> **性质**：社区/从业者总结，**非**即梦/Seedance 官方文档。  
> **画布字段**：`audioNode.data.prompt`（整段提交给 TTS API 的 `text`）。  
> **适用**：支持「指令 + 台词」一体的中文语音模型；OpenAI `tts-1` 等仅音色枚举的 API 效果有限，见文末。

## 核心原则

不要只写情绪标签（「悲伤地说」「愤怒地说」）——模型缺少可执行的声学参数。

应拆成 **四层**，再附 **台词正文**：

| 层 | 写什么 | 示例要素 |
|----|--------|----------|
| 1 音色人设 | 角色 + 年龄 + 声线 | 清冷女主、25 岁女性、柔和偏低 |
| 2 语速语气 | 快慢 + 情绪克制方式 | 语速缓慢、克制隐忍、失落感 |
| 3 气息与咬字 | 呼吸、轻重、句尾 | 气息稳定、后半句渐轻、咬字清楚、句尾轻收、强忍情绪 |
| 4 台词 | 实际要说的字 | `台词：「……」` 单独一行，引号内为朗读内容 |

## 反例 → 正例（女主独白）

**不要写：**

> 女主悲伤地说。

**可改成：**

> 清冷女主音色，25 岁女性，声线柔和偏低。语速缓慢，语气克制隐忍，带有失落感。气息稳定但后半句逐渐变轻，咬字清楚，句尾轻收，像是在强忍情绪。  
> 台词：「原来，你从一开始就没有相信过我。」

要点：情绪靠 **语速、声线、气息、句尾处理** 描述，而不是一个形容词。

## 男声 / 旁白速查

| 场景 | 表演层关键词（择 2～4 个） |
|------|---------------------------|
| 纪录片旁白 | 中年男声、中速、沉稳、气息均匀、字正腔圆 |
| 反派威胁 | 男中音偏低、语速慢、咬字重、句尾下沉 |
| 少年热血 | 偏亮男声、语速快、气息足、句尾略上扬 |
| 电话失真 | 带宽受限、略带电流感、语速正常（视模型而定） |

## 与脚本节点联动

- 从镜头表同步对白时，**先**用 `description` 里的动作，**再**用本模板扩写 `audioNode.prompt`  
- 同角色跨镜：在项目圣经角色表里记「默认音色描述」，Hermes 出配音建议时保持一致

## 模型差异（重要）

| 类型 | 建议 |
|------|------|
| 中文「可表演」TTS / 部分云厂商高级音色 | 使用完整四层模板 |
| OpenAI 兼容 `tts-1` + voice 枚举 | 主要靠 **voice**（alloy 等）+ 略缩短台词；表演层可保留作备注，但可能被整段朗读 |
| 仅朗读纯文本的 API | 表演层写入无效 → 改文案断句（逗号、省略号）模拟停顿 |

## Hermes 用法

- 侧栏问：「帮我把这句台词改成适合 TTS 的表演提示词」→ 应检索本文并输出四层结构  
- 画布：选中 `audioNode` 后对话，Brain 角色为「声音设计师」
