# 第一篇 入门与基础设施

# 第一章 产品简介与标准工作流

## 1.1 软件简介

CanvasFlow AI Studio 是一款面向内容创作者的 **跨平台桌面创作工具**。软件将 **无限画布**、**节点化工作流** 与 **AI 生成能力** 结合，使用户在同一工作空间中完成：创意输入 → 分镜脚本 → 分镜图片 → 分镜视频 → 配音配乐 → 剪辑导出。

本软件采用 **本地优先** 设计：工程文件与媒体资产保存在用户指定的本地目录；模型 API 密钥加密保存在操作系统凭据管理中，不写入工程文件。

## 1.2 六种节点类型

| 节点 | 主要功能 |
|------|----------|
| 文本 | 文本输入与 AI 对话生成 |
| 脚本 | 结构化分镜镜头表编辑 |
| 图片 | 文生图、图生图、多图融合、图像编辑 |
| 视频 | 文生视频、图生视频、首尾帧等多种模式 |
| 音频 | 文字转语音（TTS） |
| 剪辑 | 时间线编排与本地导出成片 |

各节点通过连线组成工作流，上游输出可作为下游输入。

## 1.3 标准工作流一览

典型制片链路如下：

```
[文本] ──→ [脚本] ──→ [图片] ──→ [视频] ──→ [剪辑] ──→ 导出成片
                              ↘
                               [音频]（配音/配乐，可选）
```

**操作步骤摘要：**

| 步骤 | 界面入口 | 预期结果 |
|------|----------|----------|
| 1 | 顶栏工程菜单 → 新建工程… | 本地生成工程配置文件 |
| 2 | 左侧「+ 添加」→ 文本 / 脚本 | 画布出现对应节点 |
| 3 | 连线：文本 → 脚本，填写主题后生成 | 分镜表出现镜头行 |
| 4 | 脚本 → 图片 → 视频 连线并分别生成 | 工程素材目录写入图片与视频 |
| 5 | 视频 → 剪辑 连线，单击剪辑节点 | 打开全屏「视频剪辑」工作台 |
| 6 | 剪辑台「导出」 | 剪辑节点显示成片预览 |

> **【截图占位 · 图 1-1】** 标准工作流画布俯视图（六种节点 + 连线，顶栏显示工程名）

## 1.4 主要功能模块

| 模块 | 说明 |
|------|------|
| 无限画布 | 缩放、平移、框选、小地图、对齐吸附、节点连线 |
| 六种创作节点 | 左侧添加面板、锚点菜单、节点预览与参数底栏 |
| 工作流执行 | 单节点及上游依赖执行、全图拓扑顺序执行 |
| 剪辑与导出 | 全屏剪辑工作台、时间线编辑、本地视频合成导出 |
| 工程管理 | 新建/打开/保存、多标签画布、素材库 |
| 设置系统 | 常规、模型、Agent、画布、关于 |

## 1.5 适用对象

短视频创作者、影视前期与分镜设计人员、AI 辅助内容创作用户，以及需要将创作步骤节点化、可复用的专业用户。

# 第二章 运行环境、安装与首次配置

## 2.1 硬件与系统要求

| 项目 | 最低配置 | 推荐配置 |
|------|----------|----------|
| 处理器 | 双核 2.0 GHz | 四核 3.0 GHz 及以上 |
| 内存 | 8 GB | 16 GB 及以上 |
| 硬盘 | 2 GB 可用 | 10 GB 及以上（含工程与生成资产） |
| 显示器 | 1280×720 | 1920×1080 及以上 |
| 网络 | 使用 AI 生成时需联网 | 稳定宽带 |

**支持系统：** Windows 10/11（64 位）、macOS 11+、Linux（deb/rpm/AppImage）。

## 2.2 安装步骤

### Windows

1. 运行 `CanvasFlow AI Studio_0.5.0_x64-setup.exe`（或 `.msi`）。
2. 按向导完成安装，从开始菜单或桌面快捷方式启动。
3. 防火墙提示时允许网络访问（调用模型 API 需要）。

### macOS

1. 按芯片下载对应 `.dmg` 安装包。
2. 拖入「应用程序」文件夹启动。
3. 若提示未验证开发者，在「安全性与隐私」中选择「仍要打开」。

### Linux

- Debian/Ubuntu：`sudo dpkg -i CanvasFlow-AI-Studio_*_amd64.deb`
- 或使用 AppImage：`chmod +x *.AppImage && ./CanvasFlow-AI-Studio_*.AppImage`

> **【截图占位 · 图 2-1】** 安装完成后的首次启动窗口

## 2.3 首次启动引导

首次启动弹出 **「欢迎使用 CanvasFlow AI Studio」** 对话框，说明：

- 本软件是节点化 AI 创作工作台；
- 开始创作前需在设置中配置模型 API；
- API Key **仅保存在本地设备**，不上传云端。

操作：**「前往配置」** 打开设置并进入「模型」；**「稍后配置」** 关闭引导。

> **【截图占位 · 图 2-2】** 首次启动引导对话框

## 2.4 配置 AI 模型（必做）

**步骤 1：** 点击左侧工具栏 **「设置」**。

**步骤 2：** 左侧选择 **「模型」**，顶部子标签为：

| 子标签 | 配置内容 |
|--------|----------|
| 总览 | 各类型模型就绪状态、扫码登录入口 |
| 文本与脚本 | 对话类大语言模型 |
| 图片 | 图像生成模型 |
| 视频 | 视频生成模型 |
| 语音 | 语音合成（TTS）模型 |

**步骤 3：** 在对应标签页填写服务商名称、接口地址、模型名称、API Key。

**步骤 4：** 点击 **「测试连接」** 验证；点击设置面板底部 **「保存」**。API Key 写入系统凭据管理器，**不写入** 工程文件。

**步骤 5（可选）：** 在「总览」使用 **扫码登录** 完成本地图像/视频生成账号授权（界面显示为「即梦 CLI」相关入口）。

> **【截图占位 · 图 2-3】** 设置 → 模型 → 文本与脚本（API Key 打码）  
> **【截图占位 · 图 2-4】** 设置 → 模型 → 总览

## 2.5 创建第一个工程

1. 点击顶栏 **工程菜单**。
2. 选择 **「新建工程…」**，指定空文件夹为工程根目录。
3. 顶栏显示工程名称后，「从图库选择」变为可用。

# 第五章 节点通则：新建、连线与执行

## 5.1 六种节点总览

| 序号 | 节点 | 核心能力 |
|------|------|----------|
| 1 | 文本 | 手写与 AI 对话生成 |
| 2 | 脚本 | 分镜表、全屏表格编辑 |
| 3 | 图片 | 文生图 / 图生图 / 多图融合 / 图像编辑 |
| 4 | 视频 | 多模式视频生成与预览 |
| 5 | 音频 | 文字转语音 |
| 6 | 剪辑 | 时间线编排与导出 |

## 5.2 连线规则

从 **输出锚点** 拖至 **输入锚点**：

| 上游 | 可连下游 |
|------|----------|
| 文本 | 脚本、图片、视频 |
| 脚本 | 图片、视频 |
| 图片 | 图片、视频 |
| 视频 | 视频、剪辑 |
| 音频 | 剪辑 |

## 5.3 执行工作流

| 场景 | 操作 |
|------|------|
| 执行选中节点及上游 | 选中单节点后 `Ctrl + Enter`，或点击底栏「生成」 |
| 执行全图 | 无选中时 `Ctrl + Enter` |

执行过程记录于工程运行日志，节点状态在画布回显。

## 5.4 节点通用界面

1. **节点预览区**：显示生成结果或摘要，外置标题（如「图片」「剪辑」）。  
2. **参数底栏**：选中单节点后，预览区下方浮出参数面板。  
3. **双击节点**：视口居中；脚本、剪辑可进入全屏编辑。

> **【截图占位 · 图 5-1】** 图片节点：预览区 + 参数底栏同时可见

## 5.5 右键菜单

节点或空白处右键：复制、粘贴、删除、从图库选择、保存到素材库（人物/场景/物品/风格）、对齐整理（多选时）。视频节点右键可 **创建或聚焦剪辑节点**。

## 5.6 撤销说明

画布 `Ctrl + Z` 撤销布局与连线；剪辑工作台内撤销仅作用于时间线，互不影响。

# 第六章 文本节点

## 6.1 功能说明

输入剧本文案与 Prompt；调用已配置对话模型生成或续写；作为脚本、图片、视频的上游文本；支持 **`/`** 快捷预设、**`@`** 引用其他节点。

## 6.2 新建方式

左侧「+ 添加」→ 文本；双击空白；锚点菜单；画布空态「生文本」。

## 6.3 操作步骤

1. 选中文本节点，展开底栏对话区。  
2. 输入指令，可用 `@` 引用上游内容。  
3. 选择模型（来自设置 → 模型 → 文本与脚本）。  
4. 点击 **「生成」** 或 `Ctrl + Enter`。  
5. 结果写回节点并显示在预览区。

`Ctrl + Shift + G` 可打开固定的模型对话面板。

## 6.4 典型连线

- 文本 → 脚本：提供解析主题  
- 文本 → 图片 / 视频：作为生成 Prompt

# 第七章 脚本节点

## 7.1 功能说明

将主题或剧本文本解析为 **分镜镜头表**；支持全屏表格编辑镜号、画面、台词、时长；与下游图片/视频按镜号联动；支持参考视频、角色图关联。

## 7.2 新建方式

添加面板 → 脚本；锚点菜单 → 脚本。默认标题 **分镜脚本**。

## 7.3 界面说明

| 区域 | 内容 |
|------|------|
| 节点预览 | 分镜摘要、镜头数 |
| 底栏 | 主题输入、生成/解析、进入全屏 |
| 全屏表格 | 镜号、画面描述、时长等列，支持筛选与字段显隐 |

## 7.4 操作步骤

1. 添加脚本节点；可连接上游文本或视频。  
2. 底栏填写剧情主题或约束。  
3. 点击生成/解析，等待分镜表生成。  
4. 打开全屏表检查各镜头内容。  
5. 勾选镜头后可批量操作（以表头工具为准）。

## 7.5 典型连线

- 脚本 → 图片 / 视频：按镜号驱动生成  
- 脚本 → 剪辑：剪辑台「更多」→「从脚本镜头填充」

# 第八章 图片节点

## 8.1 功能说明

支持 **文生图**、**图生图**、**多图参考融合**、**图像编辑**；可配置画幅、分辨率、风格、生成张数；预览工具条支持下载、反推提示词等。

## 8.2 任务模式（底栏顶栏 Tab）

| 模式 | 说明 |
|------|------|
| 文生图 | 文本 Prompt 生成 |
| 图生图 | 单张参考图 |
| 多图参考融合 | 多张参考图 |
| 图像编辑 | 编辑模式（需模型支持） |

## 8.3 操作步骤：文生图

1. 添加图片节点；可选文本 → 图片连线。  
2. 选择 **文生图** Tab。  
3. 编辑 Prompt（可用 `/`、`@`）。  
4. 设置画幅、分辨率、张数。  
5. 点击 **生成**，预览区显示结果。

## 8.4 操作步骤：图生图

1. 连接上游图片节点。  
2. 底栏自动或手动切换 **图生图** / **多图参考融合**。  
3. 在参考条查看缩略图，确认后生成。

## 8.5 典型连线

图片 → 视频（图生视频、首尾帧等）；图片 → 图片（链式参考）。

# 第十一章 剪辑节点

## 11.1 功能说明

汇集上游视频（必选）、音频（可选）；**单击节点** 打开全屏 **视频剪辑** 工作台；支持时间线编辑、从脚本填充、导出成片。

## 11.2 全屏工作台结构

```
┌─────────────────────────────────────────────────────────┐
│ 标题（默认「剪辑」）              [导出▾]  [关闭]        │
├─────────────────────────────────────────────────────────┤
│                    预览区（16:9）                        │
├─────────────────────────────────────────────────────────┤
│ 撤销 │ 分割 │ 删除 │ 更多… │  时间码  │  播放控制        │
├─────────────────────────────────────────────────────────┤
│ 时间线刻度尺 + 播放头 + 片段条                           │
└─────────────────────────────────────────────────────────┘
```

## 11.3 操作步骤：打开剪辑台

1. 添加 **剪辑** 节点。  
2. 视频 → 剪辑 连线。  
3. **单击** 剪辑节点进入全屏工作台。  
4. `Esc` 或 **关闭** 退出。

## 11.4 时间线操作

| 功能 | 操作 |
|------|------|
| 移动播放头 | 点击刻度尺或拖动手柄 |
| 选中/排序片段 | 点击、拖拽片段条 |
| 删除 | 工具栏或右键 |
| 裁切 | 修剪入点/出点 |
| 分割 | 定位播放头后点剪刀 |
| 播放 | 播放按钮或空格 |

## 11.5 从脚本镜头填充

脚本 → 视频 → 剪辑 已连线且视频可解析时：工作台 **「更多」** → **「从脚本镜头填充」**，按镜号写入时间线。

## 11.6 导出成片

1. 确认时间线。  
2. 顶栏 **「导出」**，选择格式与编码参数。  
3. 完成后节点预览区显示成片。

# 第十六章 常见问题与故障排除

## 16.1 无法新建工程

**现象：** 提示需使用桌面版。  
**处理：** 安装并启动桌面安装包，勿仅用浏览器预览。

## 16.2 生成无响应

检查：模型已保存且测试通过；已打开工程目录。

## 16.3 视频任务长时间等待

视频为异步任务，查看状态轨；可重启应用后继续轮询。

## 16.4 剪辑台无片段

检查：上游视频已生成；已打开工程；在节点底栏尝试刷新入片列表。

## 16.5 从脚本填充失败

检查：脚本 → 视频 → 剪辑 连线完整；分镜表有镜号；视频文件存在。

## 16.6 API Key 安全

密钥仅存本机凭据；截图说明书时须打码。
