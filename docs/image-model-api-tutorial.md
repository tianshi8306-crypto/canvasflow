# 图片模型 API 接入教程

## 1. 准备信息

在模型厂商控制台开通服务后，准备以下信息：

- API 地址（Base URL）
- API Key
- 模型名称（厂商）
- 模型型号（具体 model 值）

## 2. 在设置中新增图片模型

打开应用「设置」->「图片模型」：

1. 点击「+ 添加模型」
2. 选择「模型名称」
3. 选择「模型型号」
4. 填写「API 地址」
5. 填写「API Key」
6. 点击「保存」

> API Key 会写入系统凭据，不会保存到工程文件中。

## 3. 在图片节点使用

1. 选中图片节点
2. 在底部生成面板的「图片模型」下拉中选择刚添加的模型
3. 输入提示词并生成

生成结果会自动保存到工程 `assets/`，并回填到当前图片节点预览框。

## 4. 常见问题

- 提示「未配置图片模型 API Key」：请回设置重新填写该模型的 Key。
- 提示「图片模型未配置 API 地址」：请检查该模型的 API 地址。
- 提示模型不可用：请确认模型型号与厂商文档一致，且账号已开通该模型权限。
