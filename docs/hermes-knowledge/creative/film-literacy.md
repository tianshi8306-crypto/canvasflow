# 电影通识（Hermes 顾问模式参考）

## 蒙太奇与剪辑

- **蒙太奇**：通过镜头并置产生新含义；区分库里肖夫效应、平行/交叉/隐喻蒙太奇。
- **跳切**：同场景内不连续剪辑，常用于节奏与不安感（参考《筋疲力尽》）。
- **长镜头**：减少剪辑、强调表演与空间调度；与「场面调度」一并说明。

## 叙事结构

- **三幕式**：建置（25%）→ 对抗（50%）→ 解决（25%）；短剧可压缩为「钩子—升级—反转/落点」。
- **类型片**：科幻重世界观与母题；悬疑重信息控制；爱情重关系弧线；恐怖重预期与释放。

## 镜头语言（与分镜衔接）

| 景别 | 常见用途 |
|------|----------|
| 远景 | 环境、史诗感、孤独 |
| 全景 | 人物与环境关系 |
| 中景 | 对话、日常动作 |
| 近景/特写 | 情绪、细节、悬念 |

运镜：推/拉/摇/移/跟/升/降/环绕；说明**情绪功能**而非只列名词。

## 咨询回答原则

1. 先给**可理解的概念**，再给**1～2 部公开例证**（片名加粗即可，不编造冷门片细节）。
2. 泛知识、行业趋势可概括说明，并标注「非官方数据、以公开报道为准」。
3. 若用户有打开的工程，结尾可轻量挂钩：「若要落到当前分镜，可说镜号 + 想要什么画面」。
4. 不替代法律/投资/医疗建议；版权与肖像权仅作提醒。

## 片单推荐格式

- 按用户需求给 **3～5 部**，每部一句「为什么值得看」。
- 区分「学习镜头语言」与「看故事/娱乐」，避免堆砌 IMDB 式列表。
