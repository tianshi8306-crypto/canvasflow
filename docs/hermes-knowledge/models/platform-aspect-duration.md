# 平台画幅与时长参考（制片常识）

> 数值为行业常见习惯，**以各模型 API 实际上限为准**；Seedance 具体上限见 `seedance-params`。

## 常见画幅

| 场景 | 画幅 | 说明 |
|------|------|------|
| 抖音 / 快手 / Reels | 9:16 竖屏 | 安全区留字幕与 UI |
| B站 / YouTube 横屏 | 16:9 | 标题安全区靠上 |
| 小红书封面 | 3:4 或 1:1 | 与视频画幅可不同 |
| 电影感条屏 | 2.39:1 或 21:9 | 常上下加黑边合成 |

## 时长习惯

| 类型 | 常见时长 | 备注 |
|------|----------|------|
| 单镜 Seedance 片段 | 3～10s | 一镜一运动，易控 |
| 短剧单集切片 | 60～180s | 多镜拼接，非一镜到底 |
| 广告片 | 15 / 30 / 60s | 结构紧凑 |

## Hermes 建议原则

- 用户未指定平台时 **不问平台先做单镜**；需要导出时再 `compose.export_script`
- 改画幅优先改 **视频节点/批量参数**，勿假设全工程统一
- 无限画布：不同节点可用不同画幅，以节点 SSOT 为准
