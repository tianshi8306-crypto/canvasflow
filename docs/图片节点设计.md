# AI CanvasPro 图片节点设计规范

**核心设计理念**：预览面板与生成面板分离，节点专注于图片展示，生成参数在外部面板操作。

---

## 一、设计原则

1. **预览与生成分离**：预览面板负责图片展示，生成参数面板独立于节点
2. **预览区缩放**：仅图片预览区支持缩放查看细节
3. **操作轻量化**：提供快捷按钮、拖拽上传等高效操作
4. **状态可视化**：所有操作状态有明确的视觉反馈

---

## 二、节点结构

### 2.1 图片节点组成

```
┌─────────────────────────────────────┐
│  图片节点 n                      1920×1080│  ← 节点标题 + 分辨率
├─────────────────────────────────────┤
│                                     │
│         [ 图片预览区域 ]             │  ← 16:9 预览区，支持缩放
│                                     │
│                            [替换]   │  ← hover 显示
│                                     │
└─────────────────────────────────────┘
```

### 2.2 生成面板（独立组件）

```
┌─────────────────────────────────────┐
│ [风格] [标记]                          ↗│  ← 快捷功能栏
│                                     │
│ 描述你想要生成的内容，使用 @可引用   │  ← 提示词输入
│                                     │
│ OmniGen ▾ 16:9 ▾ 摄像机    1张  [↑] │  ← 参数工具栏
└─────────────────────────────────────┘
```

**关系**：生成面板是独立浮层，不属于节点内部，选中节点时显示在节点外部。

## 三、基础尺寸规范

### 3.1 节点尺寸

| 尺寸 | 值 | 说明 |
|------|-----|------|
| 宽度 | 280px | 固定宽度 |
| 预览区高度 | 160px (16:9) | 最小高度 260px |
| 圆角 | 10px | macOS 标准 |
| 边框 | 1px solid rgba(255,255,255,0.1) | 半透明边框 |

### 3.2 预览区

- **宽高比**：16:9
- **最小高度**：260px（AI CanvasPro 实际值）
- **支持功能**：拖拽上传、点击上传、hover 缩放

### 3.3 生成面板

- **宽度**：与节点对齐（280px 或 min(760px, calc(100vw - 56px))）
- **位置**：节点选中时显示在节点下方
- **定位方式**：使用 createPortal 渲染到 body，固定定位

## 四、预览面板功能

### 4.1 顶端功能栏

| 元素 | 说明 |
|------|------|
| 上传按钮 | 打开本地文件夹选择图片 |
| 节点名称 | 显示在左侧，可编辑 |
| 分辨率 | 显示在右侧，上传后更新 |

### 4.2 图片预览区

- **空状态**：显示占位图标 + "上传图片或输入提示词生成"
- **有图状态**：显示图片，hover 时显示替换按钮
- **交互**：
  - 点击预览区：打开文件选择器上传
  - hover 显示替换按钮
  - 支持拖拽上传

### 4.3 预览区缩放

- **Hover 缩放**：鼠标悬停时图片放大 1.35x (`transform: scale(1.35)`)
- **过渡动画**：`transition: transform 0.12s ease`
- **仅预览区缩放**：不影响生成面板

### 4.4 锚点

- **左侧输入锚点**：接收图片/文本连接
- **右侧输出锚点**：连接到其他节点

## 五、生成面板功能

### 5.1 快捷功能栏

| 按钮 | 功能 | 说明 |
|------|------|------|
| 风格 | 预设风格 | 弹出风格选择面板 |
| 标记 | 全局定位 | 画布顶部显示节点定位 |

### 5.2 提示词输入

- **placeholder**："描述你想要生成的内容，使用 @可快速引用上传的文件，按 / 呼出指令"
- **高度**：自适应，最小 60px
- **支持**：`/` 指令、`@` 引用

### 5.3 参数工具栏

| 元素 | 功能 |
|------|------|
| 模型选择 | OmniGen v2 / Seedream 5.0 等 |
| 比例 | 16:9 / 4:3 / 1:1 / 3:4 / 9:16 |
| 摄像机 | 镜头控制 |
| 数量 | 1-4 张 |
| 生成按钮 | 圆形按钮，↑ 图标 |

### 5.4 生成面板样式

```css
.image-gen-panel {
  width: 280px;
  border-radius: 10px;
  background: #1e1e24;
  border: 1px solid rgba(255, 255, 255, 0.1);
  padding: 10px;
  box-shadow: 0 4px 16px rgba(0, 0, 0, 0.25);
}
```

## 六、交互逻辑

### 6.1 节点拖拽

1. 按住鼠标左键拖拽图片预览区域
2. 拖拽时节点半透明显示
3. 显示对齐辅助线，自动吸附

### 6.2 预览区缩放

- Hover 时图片自动放大 1.35x
- 离开后恢复原大小
- 过渡平滑，不影响布局

### 6.3 素材上传

1. **点击上传**：点击预览区或顶部上传按钮
2. **拖拽上传**：拖拽图片到预览区
3. **替换素材**：hover 预览区后点击"替换"按钮

### 6.4 生成流程

1. 输入提示词或上传参考图
2. 选择模型和参数
3. 点击生成按钮
4. 生成中：预览区显示进度
5. 生成完成：预览区显示结果

## 十、实现说明

1. **预览节点**：`ImageNode` / `UnifiedImageNode` - 节点内部组件
2. **生成面板**：`ImageGenerationPanel` - 独立浮层组件
3. **定位方式**：使用 `createPortal` + `getBoundingClientRect` 跟随节点位置
4. **锚点组件**：使用 `SimpleAnchors`（左侧输入/右侧输出）
