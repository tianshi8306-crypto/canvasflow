# 稳定封板验收清单（2026-04）

> 目标：对当前「连线禁用一致性 + 交互稳定 + 可维护重构」做封板验收。  
> 建议在桌面端执行（`npm run tauri dev`），按顺序勾选。

## A1 封板验收（必过）

### 1) 连线禁用一致性

1. [ ] 右键单条连线可禁用/启用，样式切换为灰色虚线/恢复默认。
2. [ ] 多选连线后右键禁用/启用，状态提示数量正确。
3. [ ] 禁用连线不参与 `runWorkflow`。
4. [ ] 禁用连线不参与 `runNodeSubgraph` 与 `rerunFailedSubgraph`。
5. [ ] 禁用连线不参与脚本上游推导（图片/音频/视频“从脚本同步”逻辑）。
6. [ ] Inspector 的 `scriptBeatId` 区域弱提示与节点面板提示一致（enabled/disabled_only/none）。

### 2) 连线交互稳定

1. [ ] 锚点点击命中稳定（100% 与 150% 缩放均可）。
2. [ ] 连线端点贴边，无明显空隙与乱动。
3. [ ] 悬停连线 tooltip 显示摘要；禁用连线显示“已禁用（不参与执行/推导）”。
4. [ ] 悬停连线时上下游节点高亮，其他节点弱化；离开后恢复。
5. [ ] 右键连线菜单可“删除连线/定位上游/定位下游”。

### 3) 连接合法性与运行锁

1. [ ] 类型不匹配连接被拦截且提示可读。
2. [ ] 闭环连接被拦截且提示链路信息。
3. [ ] 运行期间禁止新增/删除/重连连线；运行结束后恢复。

### 4) 可观测与信息一致

1. [ ] RunPanel 最近运行解读可显示节点汇总与 Agent 阶段汇总。
2. [ ] RunPanel 显示“当前画布禁用连线：N”。
3. [ ] 失败节点与跳过原因信息可读。

### 5) 基础工程校验

1. [ ] `npm run test` 通过。
2. [ ] `npm run typecheck` 通过。
3. [ ] `ReadLints` 无新增错误。
