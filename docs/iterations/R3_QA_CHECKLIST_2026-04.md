# R3 回归清单（小白版）

适用范围：脚本工作台（侧栏 + 全屏表格）  
建议环境：桌面端（`npm run tauri dev`）  
目标：用最少步骤确认“能用、好用、不容易误操作”。

---

## 怎么用这份清单

- 每一条都按「操作 → 通过标准」执行。
- 看见“通过标准”里的结果，就可以打勾。
- 如果不通过，记录“哪一步、看到什么、期望什么”即可。

---

## 回归步骤（10 条）

1. [ ] **打开脚本节点并进入工作台**
  - 操作：选中脚本节点，打开侧栏脚本工作台。
  - 通过标准：能看到表格或卡片；无报错空白页。
2. [ ] **表格 / 卡片切换正常**
  - 操作：切到卡片，再切回表格。
  - 通过标准：数据不丢失，滚动位置体验自然（不会明显“跳丢”）。
3. [ ] **批量填写（内置预设）可用**
  - 操作：勾选 2 条以上镜头，展开“批量填写字段”，选景别或运镜，点击应用。
  - 通过标准：勾选行被批量写入；未勾选行不受影响。
4. [ ] **自定义收藏（小白高频）可用**
  - 操作：在批量“值”里输入/选择一个词，点“加入收藏”；再点击收藏词回填。
  - 通过标准：收藏词能立即复用；刷新后仍在（每类最多 5 条）。
5. [ ] **回车快速应用可用**
  - 操作：批量“值”输入框里按 Enter。
  - 通过标准：在“有勾选 + 值非空”时，等价于点击“应用到勾选项”。
6. [ ] **全屏表格打开与关闭**
  - 操作：点击“全屏表格”，再按 Esc 关闭；再打开一次。
  - 通过标准：能正常打开/关闭；再次打开数据一致。
7. [ ] **字段显隐工具可用且安全**
  - 操作：在全屏点“字段”，试“全部显示 / 恢复默认 / 全部隐藏”。
  - 通过标准：  
    - 能切换字段可见性；  
    - “全部隐藏”会保留镜号列；  
    - 不会出现 0 列（至少保留 1 列可见）。
8. [ ] **字段显隐与筛选有记忆**
  - 操作：设置几列隐藏 + 输入筛选词，关闭全屏后再打开。
  - 通过标准：字段显隐和筛选词都能恢复。
9. [ ] **字段弹层键盘流可用（无鼠标）**
  - 操作：字段搜索框中使用 `Tab/Shift+Tab` 或 `↑/↓` 切换项，按 `Enter` 切换显隐，按 `Esc` 清空搜索词。
  - 通过标准：键盘行为与提示一致，操作目标清楚（有高亮/定位反馈）。
10. [ ] **角色图/参考视频入格可用**
  - 操作：在全屏表格的“角色图”点“选图”，在“参考”点“选视频”。
    - 通过标准：文件可导入到 `assets`，单元格自动填入相对路径。
