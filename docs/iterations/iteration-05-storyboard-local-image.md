# 第 5 轮执行单：分镜关联本地图（对齐 Lib「分镜图」最小步）

## 本轮目标

在已有「分镜文案」基础上，为每条分镜支持从**本机选择一张图**，复制进工程 `assets/` 并写入 `storyboardShots[].imagePath`，在网格/列表/详情中预览；不调用云端文生图 API。

## 范围锁定

1. 数据：`StoryboardShot.imagePath`（工程相对路径）。
2. UI：详情弹层内「从本机选择分镜图」「移除图片」；网格卡片与列表缩略图。
3. 行为：`import_media_files` 单文件；重新 LLM 生成分镜文案时**保留**已有关联图路径。

## 本轮不做

- 文生图 / 图生图 API、批量自动出图。
- 从资产面板拖拽到分镜条目的复杂交互。

## Lib 指南对齐

- 对应能力树：**脚本 → 分镜图**链路中的「结果落位与回看」最小子集；完整 Lib 网格批量出图仍属后续轮次。

## 手工验收

1. `npm run tauri dev`，打开工程，脚本节点有至少 1 条分镜文案。
2. 打开某条「全文」，选一张本地 PNG，确认状态栏成功且弹层大图可见。
3. 保存工程重开，`imagePath` 仍在；再「为勾选生成分镜」，确认图片不被冲掉。
4. 列表视图「图」列有缩略图。

## UI/UX

- **关键界面**：分镜详情弹层、网格卡片顶图、列表图列。
- **关键状态**：未打开工程时按钮提示；无 `path` 时提示用桌面端。
- **本轮无** 新全局快捷键。

## 风险与回退

- 风险：浏览器预览无 `File.path`。
- 回退：移除 `storyboardShots` 中的 `imagePath` 字段相关提交。
