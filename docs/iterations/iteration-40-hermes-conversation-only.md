# Iteration 40 — Hermes 对话唯一入口（零点击执行）

- **层**：ProductionFlowLayer
- **目标**：Hermes 不提供制片控制台；用户用自然语言委托，系统自动执行；用户只在画布审核结果，用对话或节点手改迭代。

## 变更范围（3 模块）

| 模块 | 改动 |
|------|------|
| `HermesSidebar.tsx` | 识别到计划即自动 `executeDirectorPlan`；进度仅写入对话气泡；移除计划卡、参考素材条、任务轨、制片卡等 UI |
| `hermesDirectorPrefs` + `SettingsPanel` | 默认行为改为始终自动执行；设置页去掉「导演模式 / 低风险自动执行」开关 |
| `docs/product/HERMES.md` | 写入对话唯一入口原则；形态改为画布浮窗 + 灵体 |

## 功能清单

1. 有 Director 计划时**立即执行**，不展示待确认计划卡与「执行计划」按钮。
2. 「记住：…」整理后**自动入库**，不再弹出确认卡。
3. 浮窗/侧栏 Shell **仅保留**对话区 + 输入区 + 收起（侧栏形态与浮窗同构，无制片块）。
4. 产品文档与 `CURRENT_PROGRESS` 对齐本原则。

## 非目标（本轮不做）

- LLM 规划覆盖率提升、失败自动重试策略（iter-41+）
- 设置页计划模板改为纯对话管理
- 删除 `hermesTaskStore`（仍供灵体 Orb / 批量进度内部使用，不展示任务轨）
- 语音输入、宽屏展开

## 验收步骤

1. 打开工程 → 展开 H 浮窗 → 输入「帮我看看画布状态」或「分镜出关键帧」→ **不出现**计划卡按钮，对话中出现步骤列表后自动出现 `▶/✓/✗` 进度行。
2. 输入「记住：出夜景镜头时默认冷色调高对比」→ **无**确认卡，对话显示已记住。
3. 浮窗内**无**参考素材条、任务轨、创作阶段标签、圣经条。
4. 设置 → 常规：**无**「导演模式」「低风险自动执行」开关，有简短说明。
5. `npm run typecheck` 通过。

## UI/UX

- **关键界面**：画布内 Hermes 浮窗（9:16）、Idle 灵体。
- **关键状态**：规划中 / 执行中 / 流式生成 — 仅底栏一行 status + 对话内进度。
- **用户审核点**：画布节点与成片结果，非 Hermes 面板按钮。

## 风险与回退

- **风险**：误触发大批量出图/视频扣费。
- **触发**：用户未预期即开始长时间批量任务。
- **回退**：恢复 `submitUserMessage` 中 `pendingPlan` + 计划卡 UI；`defaultHermesDirectorPrefs` 改回 `directorMode: false`；恢复设置开关。
