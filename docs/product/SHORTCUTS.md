# CanvasFlow 画布快捷键

> 与 [`src/App.tsx`](../../src/App.tsx)、[`src/components/canvas/CanvasFlowChrome.tsx`](../../src/components/canvas/CanvasFlowChrome.tsx) 行为一致。  
> 产品对齐：LibTV《无限画布》§6.4；本地采用 **节点外浮层生成器**，无底部 Dock。

## 画布视图

| 快捷键 | 功能 |
|--------|------|
| `F` | 聚焦当前选中的节点（适应视口） |
| `Z` | 适配画布上全部节点到视口 |
| `Alt+Shift+F` | 整理画布（适配全部节点，并显示「还原 / 保留」） |
| `Alt+Shift+M` | 显示 / 隐藏小地图 |
| `Alt+Shift+S` | 开关节点对齐吸附 |
| 滚轮 | 在画布上缩放 |
| `Space` + 左键拖拽 | 平移画布 |
| 双击空白 | 打开添加节点面板 |
| 双击节点 | 将该节点居中到视口 |

## 选择与编辑

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` / `⌘Z` | 撤销 |
| `Ctrl+Shift+Z` / `⌘⇧Z` 或 `Ctrl+Y` | 重做 |
| `Delete` / `Backspace` | 删除选中节点或连线（不在输入框内时） |
| `Ctrl+C` / `⌘C` | 复制选中 |
| `Ctrl+V` / `⌘V` | 粘贴 |
| `Ctrl+G` / `⌘G` | 将选中节点打组 |
| 空白拖拽 / `Shift` 点选 | 框选、多选 |

## 节点生成面板

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+G` / `⌘⇧G` | 单选文本节点时钉住并打开模型对话底栏 |

有正文单选时，节点上方顶栏还提供：**复制正文**、**展开编辑**、**粘贴导入**、**下载 .txt**（图标，无长文案条）。

| 操作 | 说明 |
|------|------|
| `/` | 在图片 / 文本节点提示词输入框中呼出 Slash 预设 |
| `@` | 在提示词中引用画布内其它节点（写入正文，不替代连线参考图） |

## 说明

- Mac 上 `Ctrl` 均可用 `⌘` 替代（与系统习惯一致处见应用内快捷键浮层）。
- 脚本全屏表格、设置面板等模态打开时，部分画布快捷键会被输入焦点拦截。
