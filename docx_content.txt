第1章  软件概述
1.1  软件简介
CanvasFlow AI Studio 是一款融合了无限画布、AI 创作灵体与节点工作流的跨平台桌面 AI 视频创作工具。用户可在无限画布上自由组织灵感，通过节点连线驱动 AI 自动生成剧本、分镜、图片和视频，最终实现一键导出成片。该工具基于 Tauri 2 框架构建，兼顾轻量级桌面体验与强大的 AI 多模态生成能力。
1.2  主要功能
无限画布：基于 React Flow，支持自由拖放、节点连线与分组，提供 6 种节点类型。
AI 创作灵体 Hermes：画布内置创作助手，支持自动执行与确认执行两种模式。
节点工作流：支持文本、图片、视频、脚本、剪辑、音频共 6 种节点。
风格库：内置风格预设，一键注入生成节点。
多模态生成：文生视频、图生视频、文生图、图生图、TTS 语音合成等。
即梦 Seedance 2.0 集成：深度集成字节跳动即梦平台，支持图片与视频高质量生成。
FFmpeg 本地合成：剪辑台 + 时间线，支持 BGM 叠加，一键导出成片。
设置系统：模型配置面板，API Key 安全存储于系统凭据管理器。
本地工程管理：目录式保存，含 canvasflow.json、assets 目录与 runs.db 数据库。
自动更新：集成 Tauri Updater，开箱即用的自动升级能力。
1.3  适用范围
CanvasFlow AI Studio 适用于以下场景：
个人创作者：快速将创意转化为分镜脚本和视频成品。
短视频团队：批量生成不同风格的营销视频内容。
AI 影视研究者：实验多模态 AI 生成管线，探索节点工作流的可能性。
教育与培训：可视化学习 AI 视频生成流程与节点编排逻辑。

第2章  安装与启动
2.1  下载与安装
2.1.1  Windows 系统
访问 GitHub Release 页面：https://github.com/tianshi8306-crypto/canvasflow/releases
下载最新版本的 Windows 安装包（.msi 或 .exe）。
双击安装包，按照安装向导提示完成安装。
首次运行前，需确保 FFmpeg 已安装并添加至系统 PATH 环境变量。
2.1.2  macOS 系统
下载 .dmg 安装包。
打开 .dmg 文件，将 CanvasFlow AI Studio 拖入应用程序文件夹。
首次启动时，需在系统偏好设置中允许运行来自非 App Store 的应用。
终端中执行 xattr -d com.apple.quarantine /Applications/CanvasFlow\ AI\ Studio.app（若提示）
2.1.3  Linux 系统
下载 .AppImage 或 .deb 包。
AppImage 方式：chmod +x CanvasFlow*.AppImage &amp;&amp; ./CanvasFlow*.AppImage
Deb 包方式：sudo dpkg -i canvasflow_*.deb
确保系统已安装 GTK3 和 WebKitGTK 支持。
2.2  首次启动
安装完成后，启动 CanvasFlow AI Studio。首次启动时，软件会引导用户完成以下初始化步骤：
设置工作区目录（工程文件的默认存储路径）。
配置 FFmpeg 路径（若未在系统 PATH 中检测到）。
在设置面板中配置 AI API Key。
完成初始化后，进入主界面。
2.3  界面总览
主界面由以下区域组成：
2.3.1  顶栏（TopBar）
位于界面顶部，包含工程菜单、视图控制、Hermes 开关与设置入口。
2.3.2  画布区（Canvas）
中央主区域，基于 React Flow 渲染。用于放置节点、节点连线、缩放与平移操作。
2.3.3  左侧面板（Left Panel）
包含节点列表（+ 面板）和风格库入口。点击对应节点类型即可在画布上新建节点。
2.3.4  Hermes 面板（Hermes Panel）
显示 AI 创作灵体 Hermes 的对话与执行状态。
第3章  工程管理
3.1  新建工程
点击顶栏「工程」菜单，选择「新建工程」。
在弹出的文件选择器中，选择一个空的文件夹作为工程目录。
工程目录将自动创建 canvasflow.json 配置文件与 assets 子目录。
3.2  打开工程
点击顶栏「工程」菜单，选择「打开工程」。
选择包含 canvasflow.json 的文件夹，加载已有工程。
3.3  保存工程
3.3.1  手动保存
按 Ctrl+S（Windows/Linux）或 Cmd+S（macOS）保存工程。标签栏显示「未保存」圆点时提示需要保存。
3.3.2  自动保存
软件默认每隔 5 分钟自动保存一次工程状态。自动保存可有效防止因意外退出导致的工作丢失。

第4章  画布操作基础
4.1  画布缩放与平移
缩放：鼠标滚轮缩放画布。
平移：按住鼠标中键拖拽，或按住空格键并拖拽鼠标左键。
画布全览：按 Home 键显示全部节点。
4.2  创建节点
双击画布空白处：在弹出菜单中选择节点类型。
左侧+面板：点击「+」按钮，展开节点类型列表，选择要创建的节点。
4.3  选中、拖动与删除节点
选中节点：单击节点即可选中。
多选节点：按住 Shift 键单击可多选，或框选。
拖动节点：选中节点后，按住鼠标左键拖动至目标位置。
删除节点：选中节点后，按 Delete 或 Backspace 键删除。
节点属性：双击节点打开节点属性编辑面板。
4.4  连线操作
连接：鼠标悬停节点输出端（右侧圆点），按住拖拽至目标节点的输入端（左侧圆点），完成连线。
断开连线：单击连线，按 Delete 键删除。
循环检测：禁止创建会导致循环的连线（系统自动拦截）。
4.5  撤销与重做
撤销：Ctrl+Z（Windows/Linux）或 Cmd+Z（macOS）。
重做：Ctrl+Y 或 Ctrl+Shift+Z（Windows/Linux）或 Cmd+Shift+Z（macOS）。
第5章  节点详解 — 文本节点
5.1  创建方式
方法一：双击画布空白处 → 选择「文本」。
方法二：点击左侧「+」→「文本」。
5.2  文本输入与编辑
文本节点内部提供编辑区，支持直接输入自由文本内容作为工作流的 prompt 素材。
5.3  连接到下游节点
文本节点作为工作流起点，通过输出端口连接到下游节点，为 AI 生成提供 prompt 素材。
第6章  节点详解 — 图片节点
6.1  创建方式
方法一：双击画布空白处 → 选择「图片」。
方法二：点击左侧「+」→「图片」。
6.2  生成模式
文生图（Text-to-Image）：输入文本 prompt，AI 生成对应图片。
图生图（Image-to-Image）：基于输入图片，AI 在保持主体结构的基础上生成新图片。
6.3  参考图引用
支持在 prompt 中引用本地图片作为参考。
6.4  风格注入
从风格库面板中选中一个风格预设，点击「注入」按钮，风格参数将自动填入图片节点的风格字段。
第7章  节点详解 — 视频节点
7.1  创建方式
方法一：双击画布空白处 → 选择「视频」。
方法二：点击左侧「+」→「视频」。
7.2  生成模式
文生视频（Text-to-Video）：输入文本 prompt，AI 生成对应视频。
图生视频（Image-to-Video）：上传一张图片，AI 基于图片内容生成动态视频。
首尾帧（First-Last Frame）：同时提供首帧和尾帧图片，AI 生成从首帧过渡到尾帧的连续视频。
7.3  参数设置

时长

7.4  状态显示
视频节点显示生成状态：待机、等待中、生成中、完成、失败。
7.5  工具栏操作
提取音频：将视频中的音轨提取为独立音频文件。
裁剪：设置入点与出点，裁剪视频长度。

第9章  节点详解 — 剪辑节点
9.1  创建方式
方法一：双击画布空白处 → 选择「剪辑」。
方法二：点击左侧「+」→「剪辑」。
9.2  剪辑台界面
双击剪辑节点可打开剪辑台。剪辑台分为视频预览区、时间线编辑区与工具栏。
9.3  时间线操作
9.3.1  添加片段
从其他节点的输出端口拖拽连线至剪辑节点，自动将对应视频/音频添加至时间线。
或点击「添加片段」按钮，手动选择本地文件添加。
9.3.2  删除片段
在时间线上选中目标片段，按 Delete 键删除。
9.3.3  拖拽排序
按住鼠标左键拖动时间线条目至目标位置，松开即完成排序。
9.3.4  裁剪片段
将鼠标悬停在片段边缘，出现裁剪手柄，拖拽以调整入点/出点。
9.4  从脚本填充
点击「从脚本填充」按钮，连接至脚本节点。系统自动解析分镜数据，按照镜序顺序将每个镜头的输出视频填入时间线对应位置。此操作需要手动确认。
9.5  BGM 叠加
点击「添加 BGM」按钮，选择本地音频文件。
BGM 轨道独立于视频轨道存在，可设置音量。
9.6  导出成片
确认时间线编排无误后，点击「导出成片」按钮。
在弹出对话框中设置输出文件名、分辨率等参数。
点击「开始导出」，FFmpeg 在后台执行本地拼接与编码。
导出完成后，输出文件保存至工程 assets 目录。

第11章  Hermes 创作灵体
11.1  灵体概念
Hermes 是画布内置的 AI 创作助手，以浮动面板形式集成于主界面中。Hermes 不仅能回答用户问题，还能感知当前画布的节点布局与内容，自动理解上下文并提供创作建议。
11.2  自动执行 / 确认执行模式
自动执行模式：Hermes 识别到明确的执行意图后，自动触发相关节点，无需用户确认。
确认执行模式：Hermes 生成执行建议后暂停，等待用户确认后再执行。
模式切换：在 Hermes 面板中点击切换按钮。
11.3  对话面板操作
发送消息：在底部输入框输入文字，按 Enter 发送。
清空对话：点击「清空」按钮重置对话历史。
11.4  画布上下文感知
Hermes 持续监听画布状态变化，提供以下建议：
缺失节点建议：检测到工作流断点时，建议添加中间节点。
参数优化建议：根据生成结果质量，给出参数调整建议。
风格匹配建议：基于当前节点内容，推荐匹配的画布风格预设。

第15章  常见问题与故障排除
15.1  安装与启动问题
Q1：Windows 上安装后无法启动，提示缺少 DLL？
A：请确保已安装 Visual C++ Redistributable（推荐 VS2022）。若仍无法启动，请下载完整安装包重新安装。
Q2：macOS 提示「无法打开，因为来自未识别开发者」？
A：前往「系统偏好设置 → 安全性与隐私 → 通用」，点击「仍要打开」，或执行命令：xattr -d com.apple.quarantine /Applications/CanvasFlow\ AI\ Studio.app
15.2  工程与保存问题
Q3：修改节点后未出现未保存标记？
A：某些轻微调整不会触发标记。如需强制保存，请按 Ctrl+S 手动保存。
15.3  AI 生成问题
Q4：视频/图片生成失败？
A：常见原因包括：API Key 余额不足、模型限流、本地 FFmpeg 路径配置错误、生成的临时文件磁盘空间不足。
Q5：即梦 CLI 调用失败？
A：确认即梦 CLI 已正确安装并登录（jimeng login）。
15.4  FFmpeg 导出问题
Q6：导出时提示「FFmpeg 未找到」？
A：确保 FFmpeg 已安装并加入系统 PATH。在命令行中执行 ffmpeg -version 验证。若未安装，请从 https://ffmpeg.org/download.html 下载。
